# BENESEBOXRD – Taller Mecánico
Sitio en vivo: beneseboxrd.pages.dev

El objetivo del sitio es que cualquier persona que necesite llevar su auto al taller pueda, en pocos segundos:
- Conocer los servicios disponibles.
- Entender por qué elegir el taller (garantía, repuestos originales, diagnóstico transparente, etc.).
- Ver los horarios de atención.
- Pedir turno por WhatsApp o llamar directamente.

HTML5 – Estructura y contenido del sitio.
CSS3 – Estilos y diseño responsive

El sitio está desplegado con Cloudflare Pages, conectado a este repositorio para hacer despliegue automático ante cada cambio en la rama main.
